# Rey Tran Bot — Claude Cowork Project Package

## Mục tiêu

Package này dùng để start lại **Claude Cowork Project** cho **Rey Tran Bot** — AI General Manager của Linh Tran.

Rey Tran Bot được thiết kế để làm việc như:

- General Manager AI Agent
- Chief of Staff AI
- Executive Operating Brain
- Điều phối viên trung tâm của toàn bộ hệ sinh thái Linh Tran AI Agents
- Trợ lý vận hành qua Telegram và email
- Bộ não quản trị task, follow-up, memory, briefing, communication, decision-making
- Nền tảng để triển khai lên VPS sau này

## Cách dùng nhanh

1. Tạo Claude Project mới tên: `Rey Tran Bot — General Manager AI Agent`
2. Copy toàn bộ nội dung file `01_CLAUDE_PROJECT_INSTRUCTION.md`
3. Paste vào phần **Project Instructions** của Claude
4. Upload các file MD còn lại vào Project Knowledge
5. Khi bắt đầu chat mới trong project, dùng prompt trong file `14_FIRST_MESSAGE_TO_START_PROJECT.md`
